# AutoClick

Простой автокликер на Python, который кликает левой кнопкой мыши при нажатии клавиши.

##  Установка
1. Установите Python с сайта (если не скачан) [python.org](https://www.python.org).
2. Скачайте этот репозиторий или со страницы релизов.
3. Установите необходимые библиотеки командой в терминале:
   ```bash
   pip install keyboard mouse
   
## Использование
1. Нажимаем двойным кликом левой кнопки мыши (Или с помощью клавиши Enter)
2. Запускаем  одной клавишей (`=`).
