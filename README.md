# AutoClick

Простой автокликер на Python, который кликает левой кнопкой мыши при нажатии клавиши.

##  Установка
1. Установите Python с сайта (если не скачан) [python.org](https://www.python.org) или .exe версию.
2. Скачайте этот репозиторий или со страницы релизов.
3. Установите необходимые библиотеки командой в терминале:
   ```bash
   pip install keyboard mouse
   
## Как запустить (для пользователей)

1. Скачайте готовую версию в разделе **Releases**.
2. Запустите файл `.exe` **от имени администратора** (это нужно для работы горячих клавиш в играх).

#  Инструкция по сборке своего .exe

Если вы внесли изменения в код `AutoClick.py` и хотите создать новый исполняемый файл для Windows, который можно запускать без установки Python, следуйте этому руководству.

---

### 1. Подготовка окружения 
Откройте терминал (PowerShell или CMD) и установите библиотеки, необходимые для работы кликера и его упаковки:

```bash
pip install keyboard mouse pyinstaller
```
### 2. Подготовка файлов проекта
Для того чтобы сборка прошла успешно и файл имел красивый вид:
1.Убедитесь, что ваш основной файл называется AutoClick.py.
2.Если вы хотите иконку, положите файл icon.ico в ту же папку, где лежит скрипт.
ВАЖНО: Если вы уже пробовали собирать проект ранее, удалите папки build, dist и файл AutoClick.spec. Если этого не сделать, возникнет ошибка Permission Denied.
### 3. Работа с терминалом (путь к папке)
PyInstaller должен понимать, где лежит ваш код.
Откройте папку с проектом в Проводнике.
Кликните в адресную строку вверху, сотрите всё, впишите powershell и нажмите Enter.
Откроется терминал, который уже «смотрит» в нужную папку. Чтобы проверить это, введите команду ls — вы должны увидеть в списке свой файл AutoClick.py.
### 4. Запуск сборки
Введите финальную команду для создания одного чистого файла без лишних окон:
```powershell
pyinstaller --onefile --noconsole --icon="icon.ico" AutoClick.py
```
### Что значат эти параметры?
--onefile — собирает программу в один .exe файл (вместо папки с кучей библиотек).
--noconsole — скрывает черное окно консоли при запуске (так как у программы есть GUI).
--icon="icon.ico" — привязывает вашу иконку к файлу.
### 5. Где найти готовый результат?
После того как в терминале побегут строчки и появится надпись Completed successfully:
1.Зайдите в папку проекта.
2.Найдите новую папку dist.
3.Внутри будет лежать ваш файл AutoClick.exe.
##Решение частых проблем
###Ошибка "Permission Denied": Убедитесь, что старая версия кликера не запущена в фоне. Проверьте Диспетчер задач (Ctrl+Shift+Esc) и завершите процесс AutoClick.exe, если он там есть.
###Горячие клавиши не работают: Чтобы библиотека keyboard могла перехватывать нажатия в играх, запускайте готовый .exe файл от имени администратора.
###Антивирус блокирует файл: Поскольку файл самодельный и не имеет цифровой подписи, Windows или антивирус могут выдать предупреждение. Это нормально, просто нажмите "Выполнить в любом случае".
