# AutoClick

Простой автокликер на Python, который кликает левой кнопкой мыши при нажатии клавиши.
#  Ultra Clicker & Macro Recorder v3.3
Обновлённая версия AutoClick !
Профессиональный автокликер с графическим интерфейсом, записью макросов и защитой от зависаний.
Данное решение позволяет настраивать не прибегая к изменению кода.

##  Основные функции
- **Автокликер (Левая кнопка)**: Настраиваемый интервал до 0.001 сек.
- **Запись действий**: Полная запись движений мыши, кликов и нажатий клавиатуры.
- **Стабильный повтор**: Использование абсолютных координат (курсор не съезжает).
- **Мгновенный стоп**: Клавиша экстренной остановки всех процессов.

##  Управление
- `[ = ]` — Старт / Стоп обычного кликера.
- `[ 1 ]` — Начать запись (мышь + клавиатура).
- `[ 2 ]` — Остановить и сохранить запись.
- `[ 3 ]` — Запустить цикл повтора макроса.
- `[ 4 ]` — **ЭКСТРЕННАЯ ОСТАНОВКА** (прерывает всё мгновенно).

##  Инструкция
1. Скачайте `AutoClick.exe` из раздела **Releases**.
2. Запустите программу **ОТ ИМЕНИ АДМИНИСТРАТОРА**.
3. Установите нужный интервал кликов или количество повторов макроса.

   
## Как запустить (для пользователей)

1. Скачайте готовую версию в разделе **Releases**.
2. Запустите файл `.exe` **от имени администратора** (это нужно для работы горячих клавиш в играх).
